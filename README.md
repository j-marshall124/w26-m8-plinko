# Plinko Mod

Instructions:  
You have 6 tries to got as many points as you can!  
Move left or right using the A and D keys (or arrow keys)  
Use Space to drop a disc at the player's position

Pegs have 3 levels:  
Level 1: 10 points  
Level 2: 25 points  
Level 3: 50 points  

Well points are labelled above the corresponding well

All audio is from https://opengameart.org/
